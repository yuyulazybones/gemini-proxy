# Gemini API 的cloudflare worker 代理

使用cloudflare worker 将Gemini 2.0 免费的原生多模态API代理到国内<br>

## 使用方法
在cloudflare创建一个worker，
将 [`worker.js`](worker.js)内容拷贝到worker
再绑定一个自定义域名即可
